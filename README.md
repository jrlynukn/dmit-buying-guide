## DMIT 加量不加价到底怎么运作的

这是很多人说不清楚的地方，分开讲。

**第一层：套餐直接升级**。DMIT 在升级某个套餐规格时，不会另开新套餐让你迁移——你当前跑着的机器会同步拿到新配置。价格锁定在你购买时的水平，续费也是同一个价，没有"首年优惠陷阱"。

**第二层：硬件平台滚动升级**。这是 DMIT 最近搞的一套机制，分 Next、Stable、LTS 三个档次——

- **Next 平台**：最新款处理器，目前洛杉矶已上 AN5（AMD EPYC 9005 系列，Zen 5 架构），Pro 和 EB 系列已在迁移

当 Next 升到更新硬件，原有硬件下放 Stable；Stable 升级后，原有硬件成为 LTS。整个体系保持滚动，**你买的套餐总是跟着这个链条往前走，不会被落在最旧的硬件上**。
